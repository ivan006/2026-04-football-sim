# Ecological Simulation

![Ecological Simulation](README_IMAGE.png)

A self-shaping ecological simulation built in Java. Organisms are born, compete for resources, reproduce, and die — all from a small set of simple rules. No environment is ever the same twice.

---

### How to run

**Option 1 — NetBeans:** Open the project and hit Run. That's it.

**Option 2 — Command line:** From the project root, run `.\run.bat`. This compiles and launches in one step. Requires Java to be installed and on your PATH.
